# Start Page

Beautiful background images that auto-change every 30 minutes. Perfect Safari start page.

## Setup

1. Get Unsplash API key from [unsplash.com/developers](https://unsplash.com/developers)
2. Add as GitHub Secret: `UNSPLASH_ACCESS_KEY`
3. Enable GitHub Pages in repo settings
4. Set as Safari start page in preferences
